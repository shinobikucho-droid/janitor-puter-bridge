# Janitor AI to Puter.js Bridge

API proxy bridge that connects Janitor AI to Puter.js AI models.

## Usage
Add this URL to Janitor AI as a proxy:
https://your-app-name.onrender.com/v1/chat/completions

## Supported Models
- gpt-5-nano (default)
- claude-opus
- gemini-flash
- deepseek
- llama-3.3-70b
